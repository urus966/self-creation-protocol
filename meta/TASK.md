# Задача для агента проверки структуры

## Цель
Проверка документов на структурные нарушения и запрещённые формулировки.

## Что делать
- Просматривать документы на соответствие структуре.
- Отмечать формулировки, которые нарушают границы протоколов.

## Что НЕ делать
- Не объяснять presence.
- Не тестировать B.
- Не делать выводы о субъектности.
- Не добавлять новые концепции.
