# Vision

## Purpose

Self-Creation Protocol is evolving into a **cognitive protocol research project**.
The goal is to keep a minimal foundation where cognitive behavior can be explored safely and clearly.

## Principles

- Keep guardrails and protocol constraints explicit.
- Keep cognitive behavior modular.
- Prefer simple, inspectable flows over complex frameworks.
- Separate protocol definitions from implementation experiments.

## Outcome

The repository should act as a clean base for future `self-creation-cognitive` implementations, where protocol selection and execution can be tested without modifying guardrail sources.
